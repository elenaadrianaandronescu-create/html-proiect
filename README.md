 
# Web Application Development (HTML, CSS, JavaScript)

This project is a static web application consisting of 11 HTML pages demonstrating the core principles of web development using **HTML5**, **CSS3**, and **JavaScript**.

It includes all required elements such as different backgrounds, text formatting, lists, tables, images, internal and external links, image mapping, interactive forms, CSS animations, sound, animated GIFs, iframes, and simple JavaScript interactions.

The main goal of the project is to practice building a complete website structure and apply theoretical knowledge learned in the **“Web Application Development”** course.
